# project19-browser-template
Browser that YOU can easily create. Built with Electron and uses Chromium Engine. Brought to you by tinyurl.com/2u3fbvnd

LINUX TERMINAL IS NEEDED FOR THIS

## Features
- URL bar
- Back and forward buttons
- Reload Button
- Web Page Viewer

## Setup
1. Install Node.js (LTS Version) from node.js.org
2. Clone this repo
3. Go into the project folder by running "cd project19-browser-template"
4. Install dependencies by running "npm install"
5. Start the browser with "npm start"
